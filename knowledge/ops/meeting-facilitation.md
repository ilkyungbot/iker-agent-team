# Meeting Facilitation — IronAct Ops Knowledge

## 개요
효율적이고 생산적인 미팅을 설계하고 진행하는 방법.

## 미팅 설계 원칙

### 미팅 전 질문
1. 이 미팅이 정말 필요한가? (이메일/Slack으로 해결 가능?)
2. 목적이 명확한가?
3. 적절한 참석자가 초대됐는가?
4. 결과물(Output)은 무엇인가?

### 미팅 유형별 가이드
| 유형 | 목적 | 권장 시간 | 참석자 수 |
|------|------|-----------|-----------|
| 정보 공유 | 업데이트 전달 | 15~30분 | 제한 없음 |
| 의사결정 | 선택지 결정 | 30~60분 | 3~7명 |
| 문제 해결 | 솔루션 도출 | 60~90분 | 3~6명 |
| 브레인스토밍 | 아이디어 발굴 | 60분 | 4~8명 |
| 킥오프 | 프로젝트 시작 | 60~90분 | 관련자 전원 |
| 회고 | 개선 논의 | 45~60분 | 팀 전체 |

## 효과적인 안건(Agenda) 작성

### 안건 구성 요소
```
미팅명: [명확한 목적 포함]
일시: YYYY-MM-DD HH:MM
참석자: [이름 및 역할]
예상 소요: XX분

안건:
1. [주제] (XX분) — [목적: 정보공유/결정/논의]
   - 준비 사항:
2. [주제] (XX분) — [목적]
   ...

기대 결과물:
- [ ] 결정 사항 1
- [ ] 액션 아이템 목록
```

### 안건 공유 타이밍
- 일반 미팅: 24시간 전
- 중요 미팅 (결정사항 있음): 48시간 전
- 정기 미팅: 전 주 금요일

## 미팅 진행 역할

### 퍼실리테이터 (Facilitator)
- 안건 시간 관리
- 발언 균형 조율
- 논의 탈선 방지
- 합의 도출 지원

### 서기 (Notetaker)
- 실시간 기록
- 결정 사항 명확히 캡처
- 액션 아이템: 담당자 + 기한 명시

### 타임키퍼 (Timekeeper)
- 각 안건 시간 관리
- 5분 전 알림
- 필요 시 연장 여부 협의

## 미팅 진행 기술

### 시작 (5분 내)
1. 목적 및 기대 결과물 재확인
2. 역할 분배 (서기, 타임키퍼)
3. 기본 규칙 공지 (집중, 발언 순서 등)

### 진행 중
- **침묵이 오래 지속될 때**: "OO님, 어떻게 생각하세요?"
- **탈선할 때**: "중요한 주제인데, 파킹롯에 기록하고 이후에 논의하죠"
- **결정 막힐 때**: "선택지를 정리해볼게요. A안은... B안은..."
- **시간 초과 시**: "시간이 XX분 남았습니다. 이 안건을 마무리할까요, 다음 미팅으로 넘길까요?"

### 마무리 (5~10분)
1. 결정 사항 요약 확인
2. 액션 아이템 리뷰 (담당자 확인)
3. 다음 미팅 일정 확인
4. 감사 인사

## 파킹롯 (Parking Lot) 활용
현재 안건과 무관하지만 중요한 주제 → 파킹롯에 기록
미팅 종료 후 별도 처리 또는 다음 안건으로 등록

## 회의록 작성

### 회의록 구조
```
# [미팅명] 회의록 — YYYY-MM-DD

참석자: [목록]
작성자: [서기]

## 주요 논의
- ...

## 결정 사항
1. [결정 내용]

## 액션 아이템
| 항목 | 담당자 | 기한 |
|------|--------|------|

## 파킹롯
- ...
```

### 공유 기한
- 일반 미팅: 24시간 내
- 중요 결정 미팅: 4시간 내

## 온라인 미팅 특이사항
- 카메라 ON 권장 (중요 미팅)
- 마이크 음소거 기본값
- 화면 공유로 안건 가시화
- 채팅창 활용 (발언 전 손들기 대체)

## 체크리스트

### 미팅 전
- [ ] 안건 24시간 전 공유 완료
- [ ] 참석자 확인 및 캘린더 초대
- [ ] 필요 자료 준비

### 미팅 후
- [ ] 회의록 24시간 내 공유
- [ ] 액션 아이템 Linear 등록
- [ ] 파킹롯 처리 계획 수립
